## How to run

1. I highly recommend using `virtualenv` before continuing.

1. Install necessary **dependencies**.

  ```bash
  $ make install

  ...
  ```

2. Run **all** the tests.

  ```bash
  $ make ci

  ...
  ```

3. Run **specific** test.

  ```bash
  $  make test module=<module_name>   // ex) make test module=hello_world

  ...
  ```
